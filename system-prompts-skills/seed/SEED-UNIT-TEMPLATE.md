# SEED Unit template

Copy/paste this block into Jira and every PR description.

**Manual developer operation:** Creating/updating/merging PRs is done **manually by the developer**. This template only specifies the information that must be included.

| SEED-ID: <capability>-<sequence>  \nGoal (1-2 lines):  \nScope: (in-scope / out-of-scope)  \nConstraints: (perf, security, BC, cost)  \nAcceptance criteria:  \nEvidence required: (CI run link(s), test names, screenshots, metrics)  \nRisks & rollback:  \nSpec link: (OpenSpec spec path)  |
| :---- |
